# Carpeta de fotos

Sube aqui las imagenes que vas a usar en la invitacion.

## Rutas recomendadas

- `storage/uploads/hero.jpg`
- `storage/uploads/story-1.jpg`
- `storage/uploads/story-2.jpg`
- `storage/uploads/story-3.jpg`

## Como se usa el orden

- `STORY_PHOTO_1` se muestra primero
- `STORY_PHOTO_2` se muestra segundo
- `STORY_PHOTO_3` se muestra tercero

En el archivo `.env` debes apuntar a estas imagenes usando rutas web:

- `/uploads/hero.jpg`
- `/uploads/story-1.jpg`

No uses rutas absolutas del sistema operativo dentro del `.env`.
