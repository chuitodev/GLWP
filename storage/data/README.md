# Carpeta de respuestas

Aqui se guardan automaticamente las confirmaciones RSVP enviadas desde la invitacion.

Ruta actual: `storage/data/`

## Archivos generados

- `rsvp-submissions.txt`: formato legible
- `rsvp-submissions.ndjson`: formato estructurado por linea
- `guest-portal.json`: store principal del portal de invitados
- `guest-rsvp-history.ndjson`: historial append-only de respuestas personalizadas
- `guest-send-jobs.ndjson`: cola tecnica append-only para integracion con orquestadores
